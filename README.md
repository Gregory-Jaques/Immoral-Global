# Immoral Global

Un proyecto web moderno desarrollado con HTML, CSS y Tailwind CSS.

## Descripción

Immoral Global es una página web que presenta servicios de marketing con un diseño moderno y responsivo.

## Tecnologías utilizadas

- HTML5
- CSS3
- Tailwind CSS
- JavaScript
- Fuente Lexend de Google Fonts

## Estructura del proyecto

```
├── Index.html          # Página principal
├── style.css           # Estilos principales
├── package.json        # Dependencias del proyecto
├── tailwind.config.js  # Configuración de Tailwind
├── src/
│   ├── input.css       # CSS de entrada para Tailwind
│   └── output.css      # CSS compilado de Tailwind
├── imgs/               # Imágenes del proyecto
│   ├── Menues/         # Imágenes del menú
│   └── ...             # Otras imágenes
└── fonts/              # Fuentes personalizadas
```

## Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/immoral-global.git
```

2. Navega al directorio del proyecto:
```bash
cd immoral-global
```

3. Instala las dependencias:
```bash
npm install
```

## Uso

Abre el archivo `Index.html` en tu navegador web para ver el proyecto.

Para desarrollo con Tailwind CSS, puedes usar:
```bash
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
```
